
Django - Heroku Database + basic Email Login Tempalte
-------

This project is just meant to be an example/template for anyone that wants to boostrap a django project using a heroku database with basic authentication.

- Most of the work is done in the settings.py file.
- the MyUser model is an overriding model of the basic django auth model.  
- static assets are served with AWS S3.
